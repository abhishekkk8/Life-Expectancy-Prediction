# Life Expectancy Prediction

Machine Learning project to predict life expectancy using healthcare and economic indicators.

## Algorithms Used

- Random Forest Regressor
- Gradient Boosting Regressor
- AdaBoost Regressor

## Features

- Data Cleaning
- Missing Value Handling
- Feature Scaling
- Model Comparison
- Visualization
- Feature Importance

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results

Random Forest gave the best performance.
